# Netflix Data Analysis (Python)

## 📊 Project Overview
This project analyzes the Netflix dataset to uncover trends in content type, genres, countries, and growth over time using Python.

## 🚀 Key Features
- Movies vs TV Shows analysis
- Content growth over years
- Top countries producing content
- Most popular genres analysis

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Key Insights
- Movies are more than TV shows on Netflix  
- USA produces the highest amount of content  
- Content increased significantly after 2015  
- Drama and International genres are most common
